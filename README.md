# Peer Tutoring SaaS

## 📖 Overview
Peer Tutoring SaaS is a **Software-as-a-Service (SaaS) platform** designed to connect students with verified tutors. It offers **real-time booking, secure payments, in-app messaging, and automated scheduling** to streamline the tutoring experience.

## 🚀 Features
- **Student & Tutor Management** – Role-based access and authentication
- **Booking & Scheduling** – Real-time tutor availability, session reminders
- **In-App Messaging** – Secure communication between students and tutors
- **Payment Integration** – Supports PayPal, Stripe, and credit card payments
- **Admin Dashboard** – User analytics, tutor verification, and reporting
- **AI-Powered Tutor Matching** (Future Feature)

## 🛠️ Installation & Setup
### Prerequisites
- Node.js & npm
- Python & pip
- MySQL

### Clone the Repository
```sh
git clone https://github.com/indodanazwide/peer-tutoring.git
cd peer-tutoring-saas
```

### Backend Setup
```sh
cd backend
python3 -m venv venv
. venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
flask run
```

### Frontend Setup
```sh
cd frontend
npm install
npm run dev
```

## 🔧 Usage
- **Students**: Sign up, search for tutors, book sessions, and provide feedback.
- **Tutors**: Set availability, accept bookings, and conduct sessions.
- **Admins**: Manage users, oversee bookings, and ensure quality.
